# vrelease-py

<img src="icon.svg" height="240px" align="right"/>

Python wrapper for [`vrelease`][vrelease]. Install with pipenv:

```sh
pipenv install --dev vrelease-bin
```

Or pip:

```sh
pip install vrelease-bin
```

[vrelease]: https://github.com/vrelease/vrelease
